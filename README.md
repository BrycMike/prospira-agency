# Prospira | Premium Digital Product Agency

Prospira is a high-performance digital agency platform built with Next.js 15, React, Tailwind CSS, and Framer Motion. It features a premium design aesthetic, interactive components, and an AI-powered project advisor.

## Features

- **Modern Tech Stack**: Next.js 15 (App Router), TypeScript, Tailwind CSS.
- **Animations**: Fluid transitions and interactive elements powered by Framer Motion.
- **AI Project Advisor**: Integrated Genkit-powered AI tool to provide tech stack recommendations and roadmaps.
- **Interactive UI**: Magnetic buttons, custom cursors, and scroll progress tracking.
- **Service Estimator**: Real-time project cost estimation tool.
- **Firebase Integration**: Ready for Firestore and Authentication.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI**: [Genkit](https://firebase.google.com/docs/genkit)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Access the app at `http://localhost:3000`

## Agency Pages

- **Home**: Hero section with interactive visuals and brand trust marquee.
- **Services**: Detailed service breakdown and pricing estimator.
- **Portfolio**: Filterable showcase of selected works.
- **Process**: Step-by-step visual of the development lifecycle.
