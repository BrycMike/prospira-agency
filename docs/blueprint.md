# **App Name**: Prospira

## Core Features:

- AI Architecture Advisor: An AI-powered tool that analyzes a user's project requirements to recommend a tech stack and development roadmap.
- Dynamic Service Navigator: Interactive editorial-style service pages with expansion on hover and sticky detailed service highlights.
- Filterable Case Study Gallery: A high-performance masonry portfolio filterable by industry, tech stack, and service type.
- Interactive Pricing Estimator: A logical calculator tool that allows potential clients to build their project scope and see estimate ranges.
- Animated Process Timeline: A scroll-linked interactive 5-step roadmap (Discovery to Launch) with SVG path drawing and entrance animations.
- Integrated Lead System: A premium contact portal with field validation and hooks for third-party calendly and communication tool integrations.
- Project Dashboard Analytics: Visual count-up animations for project delivery stats and experience metrics.

## Style Guidelines:

- Primary color: Vivid Orange (#FF5B04) for energy; Secondary: Midnight Green (#075056) for a sophisticated creative touch.
- Background: Warm Ivory Cream (#FDF6E3) with Dark Gunmetal (#233038) for contrasting immersive sections.
- Headline font: 'Space Grotesk' (sans-serif) in weights 800-900; Body font: 'Inter' (sans-serif) for high legibility. Note: currently only Google Fonts are supported.
- Linear, geometric icons using Lucide React to maintain a clean and tech-forward appearance.
- Modern agency editorial layout featuring large 100vh hero sections and glassmorphic navigation components.
- Dynamic experience including fade-up reveals, magnetic button interactions, custom cursors, and subtle grain overlays.
